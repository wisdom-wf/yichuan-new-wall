# 宜川县域监测体系整合平台 - 部署文档

## 项目概述
本项目为宜川县域监测体系整合平台大屏展示系统，采用Vue 3 + Element Plus技术栈开发，支持7680×1080分辨率大屏显示。

## 部署环境要求

### 硬件要求
- **推荐配置**：
  - CPU: Intel i5 或同等性能以上
  - 内存: 8GB RAM 或以上
  - 显卡: 支持OpenGL 3.0以上
  - 存储: 10GB 可用空间
  - 显示器: 支持4K分辨率的显示器

### 软件要求
- **操作系统**: Windows 10/11 64位
- **浏览器**: Chrome 90+ / Edge 90+ / Firefox 88+
- **Web服务器**: Nginx 1.18+ 或 IIS 10+

## 部署步骤

### 第一步：准备部署文件
1. 将以下文件夹和文件复制到Windows服务器：
   ```
   dashboard-app/
   ├── dist/                    # 构建后的生产文件
   ├── public/images/          # 图片资源文件
   └── 部署文档.md             # 本部署文档
   ```

### 第二步：安装Web服务器

#### 方案一：使用Nginx（推荐）
1. 下载Nginx: https://nginx.org/en/download.html
2. 解压到 `C:\nginx\`
3. 配置Nginx：
   - 打开 `C:\nginx\conf\nginx.conf`
   - 修改server配置块：

```nginx
server {
    listen       80;
    server_name  localhost;
    root         C:/path/to/your/dist;  # 修改为实际路径
    index        index.html;
    
    location / {
        try_files $uri $uri/ /index.html;
        # 支持大屏所需的跨域配置
        add_header Access-Control-Allow-Origin *;
        add_header Access-Control-Allow-Methods 'GET, POST, OPTIONS';
        add_header Access-Control-Allow-Headers 'DNT,User-Agent,X-Requested-With,If-Modified-Since,Cache-Control,Content-Type,Range';
    }
    
    # 静态资源缓存优化
    location ~* \.(jpg|jpeg|png|gif|ico|css|js)$ {
        expires 1y;
        add_header Cache-Control "public, immutable";
    }
}
```

4. 启动Nginx：
   - 双击 `nginx.exe` 或在命令行运行 `start nginx`

#### 方案二：使用IIS
1. 打开"控制面板" → "程序" → "启用或关闭Windows功能"
2. 勾选"Internet Information Services"相关组件
3. 打开IIS管理器
4. 添加网站：
   - 网站名称：宜川大屏系统
   - 物理路径：选择dist文件夹路径
   - 端口：80
5. 配置默认文档：添加 `index.html`
6. 配置MIME类型支持所有静态文件

### 第三步：启动服务
1. 确保防火墙允许80端口
2. 在浏览器中访问：`http://localhost`
3. 系统应正常显示大屏界面

## 系统功能说明

### 主要模块
1. **视频监控墙**：显示县域视频监控点位分布
2. **视频会议**：展示视频会议系统状态
3. **气象云图系统**：可配置外部气象数据源URL
4. **应急资源分布**：显示应急物资和人员分布

### 特色功能
- **自适应缩放**：支持浏览器窗口大小调整，保持7680×1080比例
- **双向滚动**：支持水平和垂直滚动浏览
- **实时时间**：头部显示当前日期和时间
- **动态配置**：第三模块支持用户自定义URL数据源

### 操作说明
1. **气象云图配置**：
   - 点击第三模块右上角齿轮按钮⚙️
   - 输入有效的气象数据URL
   - 点击确认加载数据

2. **地图交互**：
   - 点击视频监控点位查看实时画面
   - 点击应急资源点位查看详细信息

## 常见问题处理

### 问题1：页面显示空白
**解决方案**：
- 检查Web服务器是否正常启动
- 确认dist文件夹路径配置正确
- 查看浏览器控制台错误信息

### 问题2：地图无法显示
**解决方案**：
- 确保网络连接正常
- 检查高德地图API是否可访问
- 确认浏览器支持相关API

### 问题3：图片资源加载失败
**解决方案**：
- 确认public/images文件夹已正确部署
- 检查Web服务器静态资源配置
- 验证图片文件权限

### 问题4：滚动功能异常
**解决方案**：
- 清除浏览器缓存
- 尝试使用Chrome或Edge浏览器
- 检查CSS样式是否正确加载
