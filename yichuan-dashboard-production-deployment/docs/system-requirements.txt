# 系统要求说明

## 硬件配置要求

### 最低配置
- CPU: Intel Core i3 或 AMD Ryzen 3
- 内存: 4GB RAM
- 显卡: 集成显卡 (支持1920×1080显示)
- 存储: 1GB 可用空间

### 推荐配置
- CPU: Intel Core i5 或 AMD Ryzen 5 以上
- 内存: 8GB RAM 或以上
- 显卡: 独立显卡 (支持4K显示)
- 存储: 2GB 可用空间

### 最佳配置
- CPU: Intel Core i7 或 AMD Ryzen 7 以上
- 内存: 16GB RAM
- 显卡: 专业级显卡 (支持多屏4K输出)
- 存储: SSD硬盘，5GB可用空间

## 软件环境要求

### 操作系统
- Windows 10 专业版 64位 (版本1903或更高)
- Windows 11 专业版 64位
- Windows Server 2019/2022

### 运行环境
- Node.js LTS 版本 (18.x 或 20.x 推荐)
- npm 包管理器 (随Node.js安装)
- 现代浏览器 (Chrome, Edge, Firefox)

### 网络要求
- 稳定的互联网连接
- 能够访问高德地图API服务
- 防火墙需开放8080端口

## 显示设备要求

### 主显示屏
- 分辨率: 6720 × 1260 像素 (推荐)
- 支持: 4K显示器拼接或专业大屏
- 刷新率: 60Hz 或更高

### 备选分辨率
- 3840 × 2160 (4K)
- 1920 × 1080 (Full HD)
- 系统支持自适应缩放

## 网络配置

### 端口要求
- 8080端口: HTTP服务端口
- 确保端口未被其他应用占用

### 带宽要求
- 最小带宽: 10Mbps
- 推荐带宽: 50Mbps
- 主要用于地图服务和图片资源加载

## 安装前检查清单

□ 确认硬件配置满足最低要求
□ 检查操作系统版本兼容性
□ 验证Node.js环境已安装
□ 确认网络连接正常
□ 检查防火墙设置
□ 准备好显示设备

## 性能优化建议

1. 使用SSD硬盘提升加载速度
2. 关闭不必要的后台程序
3. 使用有线网络连接确保稳定性
4. 定期清理浏览器缓存
5. 保持系统和浏览器更新到最新版本
