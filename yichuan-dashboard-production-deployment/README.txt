## 快速开始

### 1. 解压部署包
将整个文件夹解压到目标服务器

### 2. 启动系统
双击 `scripts/start-server.bat` 启动服务

### 3. 访问系统
在浏览器中打开: http://localhost:8080

## 主要功能
- 🎯 视频监控墙展示
- 🚨 应急资源分布监控
- 🌤️ 气象与水利监测
- 💧 土壤墒情实时监测
- 🔄 数据自动刷新 (10秒间隔)
- 📊 数值动态动画效果

## 技术特点
- 基于Vue 3 + 高德地图
- 响应式设计适配大屏
- 模块化架构便于维护
- 支持离线部署
- 跨浏览器兼容

## 注意事项
- 首次运行会自动安装http-server
- 确保8080端口未被占用
- 建议使用Chrome或Edge浏览器
- 生产环境建议配置为开机自启
