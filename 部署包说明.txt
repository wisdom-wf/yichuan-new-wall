# 宜川大屏系统 - 部署包结构

## 部署包内容清单

```
宜川大屏部署包/
├── dist/                           # 生产环境文件（已构建完成）
│   ├── index.html                 # 主页面文件
│   ├── css/                       # 样式文件
│   ├── js/                        # JavaScript文件
│   └── favicon.ico               # 网站图标
│
├── public/                        # 静态资源文件
│   └── images/                   # 图片资源
│       ├── hy.png                # 视频会议配图
│       ├── river.png             # 河道监控图片
│       ├── shuiku.png            # 水库监控图片
│       └── wuzi.png              # 物资分布图片
│
├── 大屏系统部署文档.md             # 详细部署说明文档
├── 部署包说明.txt                 # 本文件
└── 启动脚本/                      # 可选的启动脚本
    ├── start-nginx.bat           # Nginx启动脚本（Windows）
    └── nginx.conf                # Nginx配置文件
```

## 部署快速指南

### 简化部署步骤：

1. **复制文件**：
   将整个部署包复制到Windows服务器的任意目录

2. **安装Web服务器**：
   - 推荐使用Nginx（轻量级，配置简单）
   - 或使用Windows自带的IIS

3. **配置网站**：
   - 设置网站根目录指向 `dist` 文件夹
   - 确保默认文档为 `index.html`

4. **启动服务**：
   - 启动Web服务器
   - 在浏览器访问 `http://localhost`

### 最小化部署要求：
- 仅需复制 `dist` 文件夹和 `public/images` 文件夹
- 使用任何支持静态文件服务的Web服务器
- 确保80端口可用

## 技术规格
- **分辨率支持**：7680×1080（4K超宽屏）
- **浏览器兼容**：Chrome 90+, Edge 90+, Firefox 88+
- **响应式设计**：支持窗口缩放自适应
- **交互功能**：支持鼠标滚动、点击交互

## 注意事项
1. 确保服务器有网络连接（地图功能需要）
2. 图片资源文件必须与dist文件夹同级存放
3. 建议使用Chrome浏览器获得最佳体验
4. 部署前请先阅读详细部署文档
